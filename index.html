SELECTORES EN CSS

Selector etiqueta

Selecciona todos los elementos HTML que coinciden con una etiqueta específica.
Por ejemplo, con h2 colocaremos de color azul todos los encabezados h2 en el documento.

Selector de Clase

Selecciona elementos HTML que tienen una clase específica asignada. Se utiliza precediendo el nombre de la clase con un punto..
Por ejemplo, .selectorClass para personalizar la etique span.

Dando estilo a la etiqueta span con el selector Class

Selector de ID

Selecciona un elemento HTML con un ID específico. Se utiliza precediendo el nombre del ID con un signo de numeral (#).
Por ejemplo, #button seleccionaría el elemento con el ID "button".

Selectores Descendientes

Selecciona elementos anidados dentro de otros elementos. Se utiliza un espacio para indicar la relación.
Por ejemplo, ul li seleccionaría todos los elementos li que están dentro de una lista desordenada ul.

Selector Hijo Directo

Selecciona elementos que son hijos directos de otro elemento. Se utiliza el signo mayor que (>)
Por ejemplo, div > span seleccionaría todos los span que son hijos directos de la etiqueta div.

Selector Hermano siguiente o adyacente

Selecciona un elemento que es un hermano inmediato del elemento especificado. Se utiliza el signo de suma (+).
Por ejemplo, p + span seleccionaria un span que sigue inmediatamente a un parrafo.

Selector Hermanos siguientes o adyacentes

Selecciona todos los elementos hermanos inmediatos que siguen al elemento especificado. Se utiliza el signo (~).
Por ejemplo, p ~ a seleccionaría todos los enlaces que siguen a los parrafos.

Anidamiento de selectores

Permite combinar múltiples selectores para aplicar estilos a elementos que cumplen con cualquiera de los selectores combinados. Se utiliza una coma para separar los selectores.
Por ejemplo, .box1, .box2, .box3 seleccionaría dichos divs en el documento.
